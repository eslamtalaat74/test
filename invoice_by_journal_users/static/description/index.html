Account Journal Users Restriction

Filtering of Invoice / Bill / Journal Entries Restriction by Journal Users

Filtering of Debit Note / Credit Note / Sales and Purchase Receipts Restriction by Journal Users

This app allows you to restrict journals by users and also filter

Invoice / Bill / Journal Entries / Debit Note / Credit Note / Sales and Purchase Receipts for restricted journal users.

Main Features:

  • This app allows you to restrict journals by users that means if user comes under restricted group and if user is allocated only specific journals then system will filter journals and invoices, bills, credit and debit notes, journal entries, sales and purchase receipts for that users for that assigned journals only so that user will not able to see mentioned documents where journal is not available for him/her.
  • Limited Access by Journals Group on user form for configuration. So restriction will only apply to this group users only.
