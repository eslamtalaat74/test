Odoo SMS Gatways and Notification

Odoo SMS Gateway App helps the user to send SMS notification to the registered mobile number. This apps support MSG91 & ClickSend, and TextLocal SMS Gatways for sending SMS. Users can send multiple SMS to all their customers and suppliers. Users can select templates of their requirement using this app and send to their supplier(s) and customer(s).

SMS template for multiple SMS.

User can multiple SMS template for sending SMS to multiple customer(s) and supplier(s).

Configure SMS gateway

User can configure the SMS gateway, with sender name, message-authentication key.

Generate SMS history

User can generate SMS history and check it at any time.

Send SMS Campaigns

User can create a campaign for sending SMS to particular groups.

Send SMS To Multiple Members

User can select "Multiple Members" options for sending SMS to Contacts

Send SMS For Specific Action

User can send SMS for specific action like, Create Sale Order, Confirm Sale Order Or Invoice Validated.
